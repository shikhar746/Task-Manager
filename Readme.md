# Task Manager

A full-stack task management application with a CLI and Web Dashboard sharing a single SQLite database.

## Tech Stack

- **Runtime:** Node.js
- **Database:** SQLite (`better-sqlite3`)
- **CLI:** `commander`, `chalk`
- **Web Backend:** `Express`, `express-session`, `bcrypt`
- **Web Frontend:** Plain HTML, Tailwind CSS (CDN), Vanilla JS (Fetch API)
- **Testing:** `Jest`
- **CSV Export:** `csv-writer`

git clone https://github.com/yourusername/task-manager.git
